Unilever Canteen Ordering System

This project addressed the inefficiencies in Unilever's current canteen operations where employees face long waiting times, limited food availability, and food wastage resulting in productivity losses and dissatisfaction. The new system enabled employees to pre-order meals online, reduced congestion, minimized food wastage, and enhanced overall satisfaction. The proposal outlined the necessary changes, benefits, risks, and implementation plan to achieve these objectives using Stakeholder Analysis Matrix, Business Case, RACI Matrix, FishBone Diagram, Process Flow mappings, and user Stories. This project was done using Miro, Drawio, Jira, and Figma.
